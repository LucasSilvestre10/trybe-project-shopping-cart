#  Carrinho de Compras

## Descrição do Projeto
Bem-vindo ao repositório do projeto Carrinho de Compras! Este projeto atende aos requisitos propostos no curso da Trybe. A seguir, os detalhes de cada requisito:

### Requisito 1 (TDD) - Desenvolvimento de Testes para `fetchProducts`
- Foi realizado o desenvolvimento de testes para garantir no mínimo 25% de cobertura total e 100% de cobertura na função `fetchProducts`.

### Requisito 2 - Listagem de Produtos
- Utilizando a função `fetchProducts`, foi criada uma listagem de produtos através da API do Mercado Livre.

### Requisito 3 (TDD) - Desenvolvimento de Testes para `fetchItem`
- Foi realizado o desenvolvimento de testes para garantir no mínimo 50% de cobertura total e 100% de cobertura na função `fetchItem`.

### Requisito 4 - Adição de Produto ao Carrinho
- A função `fetchItem` foi implementada para retornar dados de um produto e adicioná-lo ao carrinho de compras.

### Requisito 5 - Remoção de Item do Carrinho
- Ao clicar no produto no carrinho de compras, ele é removido da lista.

### Requisito 6 (TDD) - Desenvolvimento de Testes para `saveCartItems`
- Foi realizado o desenvolvimento de testes para garantir no mínimo 75% de cobertura total e 100% de cobertura na função `saveCartItems`.

### Requisito 7 (TDD) - Desenvolvimento de Testes para `getSavedCartItems`
- Foi realizado o desenvolvimento de testes para atingir 100% de cobertura total e 100% na função `getSavedCartItems`.

### Requisito 8 - Carregamento do Carrinho de Compras ao Iniciar a Página
- Os itens adicionados no carrinho de compras são salvos no localStorage, garantindo sua persistência ao recarregar a página.

### Requisito 9 - Cálculo do Valor Total dos Itens do Carrinho
- O elemento com o valor total dos produtos possui a classe `total-price`.

### Requisito 10 - Limpeza do Carrinho de Compras
- Implementada a lógica no botão "Esvaziar Carrinho" para limpar todos os itens do carrinho de compras.

### Requisito 11 - Adição de Texto "Carregando" durante Requisição à API
- Adicionado um elemento com o texto "Carregando..." durante a requisição à API.


## Habilidades Técnicas
O projeto foi desenvolvido utilizando as seguintes linguagens e tecnologias:

- Linguagem de Programação: JavaScript
- Testes: Jest
- Frontend: HTML, CSS
- Gerenciamento de Estado: localStorage
- API: Mercado Livre API
